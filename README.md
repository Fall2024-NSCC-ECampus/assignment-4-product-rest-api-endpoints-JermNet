# How To Run
1. Have XAMPP, and an IDE (preferably InteliJ) installed on your system.
2. Start "Apache" and "MySQL" in XAMPP.
3. Create a schema at "localhost/phpmyadmin" named "advanced4".
4. Run project in your IDE.
